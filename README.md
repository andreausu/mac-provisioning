Mac Provisioning
================

Ansible playbook for the provisioning of my MacBook.
Tested on Mac OS X Mavericks 10.9

Just run:

```shell
$ ./run.sh
```

Installed software:

- Homebrew + Cask
- Ansible
- zsh
- ohmyzsh
- rvm
- git (lastest version)

- nginx
- node
- automake

- elasticsearch

- openssl
